# FrequencyOS — Anti-Swing Inbox / 摇摆冲动缓冲池

> 原则：写下来 ≠ 行动。写下来本身就是冷却的第一步。
>
> 任何"换方向 / 开新坑 / 放弃 / 焦虑 / 新鲜感"的冲动，先落到这里，**默认 24 小时不动手**。

## 协议（必读）

当出现以下任一信号时，触发拦截流程：

- 想新开一个项目 / 方向
- 想关停 / 放弃一个未到 90 天的目标
- 因别人成功 / 失败而动摇当前目标
- 因单次表扬 / 批评而想重构方向
- 因行业新闻 / 风口而想转轨
- 因短期情绪（兴奋 / 焦虑 / 厌倦）而想改目标

### 拦截流程

```
1. 写入本文件（一条登记）
2. 标注情绪强度（1-10）+ 冲动类型（5 类）
3. 默认进入 24h 冷却（什么都不做）
4. 24h 后若情绪强度 ≥ 7：延长 72h 微验证
5. 72h 内只能做信息采样，禁止开仓 / 关仓
6. 72h 后做三频评分：
   - 总分 ≥ 12 且单维 ≠ 0 → 允许升级主频候选
   - 总分 9-12 → 进入副频池
   - 总分 < 9 → Kill，写入 killed_reasons.md
```

### 五种冲动类型

| 类型 | 特征 | 处置 |
|---|---|---|
| 战略机会 | 三频同时成立、非对称上行 | 72h 微验证 |
| 高频噪音 | 来自外部热点 / 别人成败 | 24h 冷却后基本自动失效 |
| 情绪逃避 | 现实困难时想换轨 | 强制 72h 冷却 + 写 500 字回避说明 |
| 新鲜感渴求 | "新东西才有意思" | 24h 冷却 + 写下"为什么旧的不再新鲜" |
| 身份焦虑 | "我是不是该是 XX 类型的人" | 24h 冷却 + 重读 goals.md 的「主动放弃」 |
